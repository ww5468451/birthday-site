# to-iris-birthday-site

Static birthday website, reorganized into a cleaner `pages/` and `assets/` structure without changing the original content flow or animations.

Main entry:

- `index.html`

Important folders:

- `pages/`: page files
- `assets/styles/`: CSS
- `assets/scripts/`: custom JavaScript
- `assets/vendor/`: third-party libraries
- `assets/media/`: audio and images
